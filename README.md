# Crawl Walk Run Swing

A way to educate athletes on how to better their movement systems and become optimal athletes.

Single-page site: assessments (Movement Age, Sense Dominance, Which Suit), daily polls, and calculators (1-Rep Max with full 1RM-12RM chart, Body Fat & BMI) — all with the math shown, not hidden.

## Structure
- `index.html` — the entire site (HTML, CSS, JS in one file)

## Deploy
Import this repo into Vercel — no build step needed, it serves `index.html` as-is.
